# pet-adoptables-api
A REST API built using Node.js, Express.js, and PostgreSQL.

### Requirements
* Node.js
* Express
* PostgreSQL

### Installation
* Download Github repo
* Setup PostgreSQL database (SQL file include in repo)
* Run the following commands in terminal:
  * ```npm install```
  * ```npm start```

### API Endpoints

##### Users
* ```GET``` /users
* ```POST``` /users
* ```GET``` /users/:id 
* ```PUT``` /users/:id 
* ```DELETE``` /users/:id

##### Pets
* ```GET``` /pets
* ```POST``` /pets
* ```GET``` /pets/:id 
* ```PUT``` /pets/:id 
* ```DELETE``` /pets/:id 
* ```GET``` /pets/types
